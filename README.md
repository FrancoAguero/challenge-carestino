## Challenge Carestino

Se creo el proyecto con React + Vite + PNPM

### Estructura del proyecto

```bash
paint/
├─ public/
│  └─ vite.svg
├─ src/
│  ├─ components/
│  │  ├─ PaintCell/
│  │  │  ├─ PaintCell.jsx
│  │  │  └─ PaintCell.module.css
│  │  ├─ PaintGrid/
│  │  │  ├─ PaintGrid.jsx
│  │  │  └─ PaintGrid.module.css
│  │  └─ Palette/
│  │     ├─ Palette.jsx
│  │     └─ Palette.module.css
│  ├─ views/
│  │  └─ Paint/
│  │     └─ Paint.jsx
│  ├─ Index.css
│  └─ main.jsx
├─ .gitignore
├─ eslint.config.js
├─ index.html
├─ package.json
├─ pnpm-lock.yaml
├─ README.md
└─ vite.config.js
```

### instalacion

```bash
  pnpm install

  pnpm run dev
```
